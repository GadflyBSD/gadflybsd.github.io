字数总计: 325|阅读时长: 1 分钟| 阅读量:

使用七牛云的js-sdk, 开发一个vscode的上传插件.
一个 VS Code 插件，编写 Markdown 时可以快捷上传本地图片获取七牛图床外链。
只有在编辑 Markdown 时插件才可使用。

使用

  1. 粘贴图片路径上传：SHIFT + P
  2. 直接选择图片上传：SHIFT + O.
  3. 截屏图片上传： SHIFT + V

按键需在英文编辑状态下有效，功能2 需要升级 vscode 到 v1.17+。
