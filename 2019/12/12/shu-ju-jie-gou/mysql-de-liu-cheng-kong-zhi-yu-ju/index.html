字数总计: 387|阅读时长: 1 分钟| 阅读量:

  mysql 操作同样有循环语句操作，网上说有3中标准的循环方式： while 循环 、 loop 循环和repeat循环。还有一种非标准的循环： goto。 鉴于goto语句的跳跃性会造成使用的的思维混乱，所以不建议使用。这几个循环语句的格式如下：

  • 条件判断语句
    • [x] IF ... else
  • 分支语句
    • [x] case
  • 循环语句
    • [x] WHILE……DO……END WHILE
    • [x] REPEAT……UNTIL END REPEAT
    • [x] LOOP……END LOOP
    • GOTO
